过了四级，一切也就不那么简单了。音乐变得越来越快，越来越长，却很难听出有什么调子在里面。老师带我们弹练习曲，弹几页几页的乐谱，弹车尔尼巴赫之类写的，十分晦涩的古典音乐。越到后面，音乐对演奏者技法的要求就越高，因为书上的都是古典音乐。我对这类音乐的反感，就是这时建立起来的。 父亲一直站在我身后，看我坐在琴凳上练习。我靠在这座左右一望无边的山脚下，吃力地敲出每一个音符。每一个音符从乐谱中跳出来，但没有乐谱中那样的璀璨、有活力，它们在空气中机械地组成了一段毫无曲调的、毫无价值的sin曲线。我听的出来，这是段没有生机的音色；我也听的出来，父亲心中有一股失落。 这些年，我一直在找寻一个时间－－就算一天也好－－可以让我休息一下。然而，父亲的坚持，使我放弃了这方面的幻想。而到了现在，我彻底被解放了，却觉得有那么一丝丝的，说不清的若有所失的感觉。 时光正从我指间流出。 那些弹琴中的和谐场景，正一帧一帧地浮现在我眼前：父亲在我的演奏取得进展时的高兴和夸耀；我在完成一首曲子，在班里大显身手的激动与自豪；老师在看到自己学生在音乐方面有所建树，取得成绩的骄傲与感慨……这些生动的画面，使我拾起了当年弹琴的快乐时光，领悟了弹琴所带来的美好和快乐。 我所苦苦寻求的，缺失的东西，不就是这特别的快乐吗？
